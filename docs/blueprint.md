# **App Name**: GestorFácil

## Core Features:

- Registro de Transacciones: Permite a los usuarios registrar ingresos y gastos con fecha, descripción y monto.
- Clasificación por Categorías: Gestiona y asigna transacciones a categorías personalizables (ej. Alimentación, Transporte, Ocio).
- Creación de Presupuestos: Define límites de gasto mensuales o personalizados por categoría para un control efectivo.
- Dashboard Financiero: Visualiza un resumen rápido del estado financiero, gastos por categoría y progreso del presupuesto.
- Análisis de Gastos con IA: Una herramienta de IA que analiza patrones de gasto y proporciona sugerencias personalizadas para optimizar el presupuesto.
- Persistencia de Datos: Almacenamiento seguro y recuperación de todos los datos financieros del usuario en una base de datos.

## Style Guidelines:

- Esquema de color principal: Un azul profesional y tranquilizador (#2275C0), que evoca confianza y claridad en la gestión financiera. Se usará en elementos clave y acciones primarias.
- Color de fondo: Un blanco roto muy claro con un matiz sutil de azul (#EEF3F5), manteniendo un entorno limpio y minimalista para una fácil legibilidad.
- Color de acento: Un verde fresco y enérgico (#29996E), que denota crecimiento y salud financiera. Se utilizará para elementos de progreso, estados positivos y llamados a la acción secundarios.
- Fuentes para encabezados y cuerpo de texto: 'Inter', una sans-serif moderna y neutral que garantiza una excelente legibilidad de los datos numéricos y el contenido.
- Iconos limpios y minimalistas que representen conceptos financieros comunes, como cartera, gráfico, categorías de gasto y signos de moneda, manteniendo un aspecto cohesivo.
- Un diseño de panel intuitivo y responsivo, priorizando la información más importante en la parte superior y facilitando la navegación entre secciones clave.
- Transiciones y microinteracciones sutiles para mejorar la experiencia del usuario, como al guardar una transacción o cambiar de vista, sin distraer del contenido.
